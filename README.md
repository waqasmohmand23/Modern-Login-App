# Modern Login App

A modern, comprehensive Flutter login and registration application. This project demonstrates best practices for UI/UX design, state management, and form validation in Flutter.

## Features

- **Modern UI/UX**: clean, aesthetic design with smooth animations using `animate_do`.
- **Authentication Flow**: Complete Login and Registration screens.
- **State Management**: Uses `provider` for efficient state management.
- **Input Validation**: Robust validation for email and password fields.
- **Password Visibility**: Toggle to show/hide password text.
- **Custom Widgets**: Reusable components like `CustomTextField` and `GradientButton`.
- **Responsive Design**: Adapts to different screen sizes.

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

- Flutter SDK: [Install Flutter](https://flutter.dev/docs/get-started/install)
- Dart SDK (included with Flutter)

### Installation

1.  **Clone the repository:**

    ```bash
    https://github.com/waqasmohmand23/Modern-Login-App.git
    ```

2.  **Navigate to the project directory:**

    ```bash
    cd modern_login_app
    ```

3.  **Install dependencies:**

    ```bash
    flutter pub get
    ```

4.  **Run the app:**

    ```bash
    flutter run
    ```

## Dependencies

- [flutter](https://flutter.dev)

## Project Structure

- `lib/core/theme`: App theming and styling.
- `lib/providers`: State management providers (e.g., `AuthProvider`).
- `lib/screens`: Application screens (Login, Register).
- `lib/widgets`: Reusable UI components.
